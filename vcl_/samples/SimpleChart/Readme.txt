SimpleChart is a control used to show stadistical graphics in a wide range of ways.

This example shows the functionalilty of this control with the possibility of
adding captions to the chart axis and allowing the user to add
different amounts that will be then showed in the graphic.

The type of representation can be changed changing the value of combo Type.

The Apply button must be pushed after the type is changed, and the Add button
to add quantities to the graphic.
