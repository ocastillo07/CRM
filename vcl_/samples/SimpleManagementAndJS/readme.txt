In this sample you can verify how to show, delete and add data in a database
easily using components like table, datasource, database, panel, dbrepeater...

In order to run this sample, you must use the oscommerce.sql file that you can
find in /vcl/samples directory to create the necessary database. Fact this, you
must change the username and userpassword properties of database oscommerce
component to connect with the database.
