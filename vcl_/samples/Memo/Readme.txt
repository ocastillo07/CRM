Use Memo to put a standard multiline edit control on a form. Multiline edit boxes
allow the user to enter more than one line of text. They are appropriate for
representing lengthy information.

The Memo example presents some memo features like multiline suppport, text selection,
and retrieving single lines of text by index.

There are no arbitrary limitations to to how much text it can contain, but
client and server-side performance can noticeably degrade if it contains much text.
